
ECOSNAP - ANDROID IMPLEMENTATION OUTLINE
-----------------------------------------

1. PROJECT SETUP (build.gradle.kts)
   - Add CameraX dependencies (camera-camera2, camera-lifecycle, camera-view)
   - Add Coil for image loading
   - Add Google GenAI SDK (com.google.ai.client.generativeai)
   - Add Jetpack Navigation Compose

2. THEME (Color.kt)
   - Primary: #059669 (Emerald 600)
   - Secondary: #ecfdf5 (Emerald 50)
   - Background: #ffffff

3. APP ARCHITECTURE
   - Use Clean Architecture with MVVM
   - Repository: EcoRepository (handles Gemini API calls)
   - ViewModel: HomeViewModel, ScanViewModel, SwapsViewModel
